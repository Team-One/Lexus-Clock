tomtom.Animation Class

Defined in: src/main/webapp/js/tomtom/Animation.js:11

Creates an animation object which can be used to play animations for DOM Nodes.

Constructor

tomtom.Animation

(
  • element
  • effect
  • [options]
)

Defined in src/main/webapp/js/tomtom/Animation.js:11

Parameters:

  • element String | DOM Node

    The element the animation should be performed on.

  • effect String

    The effect used to play the animation. Should be one of "bounce", "dropIn", "fadeIn", "fadeOut".

  • [options] Object optional

    Additional options for the animation.

    • [duration=500] Number optional

      The duration the animation should be played for.

    • [delay=0] Number optional

      The time to wait before starting the animation.

    • [callback] Function optional

      The callback to be executed when the animation has completed.

Example:

new tomtom.Animation(document.getElementById("test"), "fadeIn").play();
