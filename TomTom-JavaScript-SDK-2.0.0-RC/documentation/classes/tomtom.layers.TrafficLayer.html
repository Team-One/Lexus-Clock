tomtom.layers.TrafficLayer Class

Extends L.TileLayer
Defined in: src/main/webapp/js/tomtom/layers/TrafficLayer.js:10

Displays traffic flow information on top of the TomTom base map. It is recommended that this class is not used directly but rather the tomtom.Map's displayTraffic option is used like this:

var map = new tomtom.Map({
    domNode: "map",
    displayTraffic: true
});
