geocode

(
  • address
  • [options]
  • callback
)

Defined in src/main/webapp/js/tomtom/services/GeocodingService.js:22

Executes a geocoding request for the specified address. This will take an address and return the latitude and longitude of the address.

Parameters:

  • address String

    The address to geocode.

  • [options] Object optional

    The options passed to the geocoding service.

    • [projection] String optional

      The projection used to specify the coordinates in the request and response. Supported values are EPSG4326 or EPSG900913.

    • [language] String optional

      The two-character ISO code for the output language. Supported languages are de, en, es, fr, it, nl.

    • [maxResults] Number optional

      The maximum number of hits for the geocoder to return.

  • callback Function

    The callback to execute when a response is received from the geocoding service.

Example:

var service = new tomtom.services.GeocodingService("API Key");
service.geocode("11 Lafayette St, Lebanon, NH", function(response) { ... });

reverseGeocode

(
  • latitude
  • longitude

  • callback
)

Defined in src/main/webapp/js/tomtom/services/GeocodingService.js:74

Executes a reverse geocoding request, which allows a latitude and longitude to be turned into an address.

Parameters:

  • latitude Number

    The latitude to look up the address for.

  • longitude Number

    The longitude to look up the address for.

  • [options] Object optional

    Additional options to pass to the geocoding service.

    • [type] String optional

      Defines the importance of the returned value: are we looking for the name of a village (Regional), or the large city it's close to (International)? Different values can result in different fields in the response, or different values for the same field. Basic (default) - Small number of fields. Contains formated address (concatenated data). All - List of results for all types together. International - Values of nearest big locations. National - Values of nearest medium locations. Regional - Values of nearest locations.

    • [projection] String optional

      The projection used to specify the coordinates in the request and response. Supported values are EPSG4326 or EPSG900913.

    • [language] String optional

      The two-character ISO code for the output language. Supported languages are de, en, es, fr, it, nl.

  • callback Function

    The callback to execute when a response is received from geocoding service.

Example:

var service = new tomtom.services.GeocodingService("API Key");
service.reverseGeocode(43.67863, -72.26259, function(response) { ... });
